# Features Catalogue

This repo is the data storage for feature reports on the [GTMC website](https://beta.techmc.wiki). Each feature report is associated with an issue under this repository.

## *"Deleting"* an Feature

For admins/reviewers, if you want to delete(instead of marking as resolved as usual), you can do this by close the corresponding issue. The feature will be hidden and marked as read-only on the website.

## Opening Features Manually

It is planned to allow creating features directly through creating an issue under this repo. Although that might lead to many management difficulties.

This method is not allow until a prope issue template were created
